# jsondiffpatch-html

CLI tool to generate HTML diff reports from JSON files using `jsondiffpatch`.

## Install Dependecies

```bash
npm install
```

## Add execute permission

```
chmod +x bin/jsondiffpatch-html.js
```

## Link the command

```
npm link
```

## Run the command

```
jsondiffpatch-html left.json right.json -o diff.html
```
